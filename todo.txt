  왼쪽  : 사용자 화면

청소하는 분들 화면

1. 지도

   1)'AI 최적 경로 추천' 아이콘 오른쪽 하단 
    
      - 자신이 서있는 위치에서 최적의 경로로 쓰레기를 가질 수 있는 방법 추천

   2) 나머지는 사용자와 동일하게

   3)쓰레기 모양 아이콘을 눌렀을 때 간단하게 정보를 알려주기
    -간단한 정보 보여주기



2. 게시글
    1)좋아요 순서에 따라 내림차순으로 게시물이 뜨게 한다.
    -좋아요가 많을수록 치울 필요성이 높다고 판단.
    2)이 게시글에는 지도로 넘어가는 기능이 필요.
    -지도로 넘어가서 현재 위치에서 쓰레기 위치까지 경로를 알려준다.
    
    3)쓰레기를 청소하기 전과 후로 사진을 변경할 수 있도록 한다.
    







1. 청소하시는 분들 위치를 알아야함
2. 쓰레기 더미 위치와 사진 찍은 위치와 가까운 사람들에게 연락을 준다.AI3
3. 좋아요를 많이 받은 사진들이 맨 위에 올라가게 한다.
3. 좋아요를 많이 받은 사진들의 위치를 알려준다.
4. 청소를 했다면 청소한 사진을 청소 전 사진과 비교해서 올리는 게시판이 있다.AI
5. 


사용자화면

1. 제보
    1)사진을 불러오는 기능
    -기존 이미지
    -실시간으로 찍은 사진 업로드
    2) 
    -
    3) 위치 불러오는 기능
    -현재 위치를 불러오는 기능
    4)게시글에 등록하는 기능

   기존의 틀을 유지 
   위에 이미지 사진을 누를 경우 (카메라 / 갤러리)

2. 지도

   1)네이버 지도 화면 표시

   2)쓰레기 모양의 아이콘 표시

   3)화면 확대: 쓰레기의 위치가 각각 보임
     화면 축소: 쓰레기의 위치가 합쳐짐 (동그라미가 숫자가 커질수록 커짐)
   
   4)쓰레기 모양 아이콘을 눌렀을 때 간단하게 정보를 알려주기
    -간단한 정보 보여주기


 3. 게시글(첫화면)

    1)스크롤로 다른 사람들이 찍은 사진을 볼 수 있다

    2)화면을 두번 누를 경우 좋아요 표시가 나타난다

    3)날짜, 좋아요, 위치의 정보가 나타난다

    4)사진 순서는 무작위로(여러 사진을 볼 수 있도록)
4. 뉴스
    환경을 생각하는 뉴스를 올려서 환경 보호에 대한 중요성에 대해서 강조합니다.

2. 인스타처럼 사진과 위치정보를 올린다.
3. 다른 사용자들이 좋아요를 누를 수 있는 기능이 있음
4. 사용자들한테는 무작위 안본 사진들이 화면 보이게 해줘야 함
